Este repositorio reúne una serie de cuadernos en **Jupyter Notebook** para aprender y practicar fundamentos de **Python** aplicados a la manipulación de datos y programación básica.

## Objetivo general
Fortalecer las bases de Python (estructuras de datos, control de flujo y programación modular/POO) para poder resolver ejercicios y construir soluciones con código claro, reutilizable y fácil de mantener.

---

## Contenido del repositorio (con objetivos)

### 01 — Cadenas (`01-Python-Cadenas.ipynb`)
**Objetivos:**
- Entender qué es una cadena (string) y cómo se manipula.
- Aplicar indexación y slicing.
- Usar operaciones comunes: concatenación, longitud, métodos y formato.

### 02 — Tuplas (`02-Python-Tuplas.ipynb`)
**Objetivos:**
- Comprender la estructura e inmutabilidad de las tuplas.
- Acceder a elementos con indexación/slicing.
- Aplicar operaciones básicas y casos de uso típicos.

### 03 — Listas (`03-Python-Listas.ipynb`)
**Objetivos:**
- Crear y modificar listas de forma práctica.
- Dominar métodos comunes (append, insert, remove, sort, etc.).
- Resolver problemas usando listas y recorridos.

### 04 — Conjuntos (`04-Python-Conjuntos.ipynb`)
**Objetivos:**
- Entender qué es un set y para qué sirve.
- Aplicar operaciones: unión, intersección, diferencia y pertenencia.
- Usar sets para eliminar duplicados y comparar colecciones.

### 05 — Diccionarios (`05-Python-Diccionarios.ipynb`)
**Objetivos:**
- Modelar información con pares clave–valor.
- Acceder, actualizar y recorrer diccionarios.
- Aplicar diccionarios en problemas de conteo, búsqueda y mapeo.

### 06 — Condiciones (`06-Python-Condiciones.ipynb`)
**Objetivos:**
- Usar `if`, `elif`, `else` para tomar decisiones.
- Aplicar operadores lógicos y de comparación.
- Construir validaciones claras para distintos escenarios.

### 07 — Bucles (`07-Python-Bucles.ipynb`)
**Objetivos:**
- Repetir procesos con `for` y `while`.
- Entender `range()`, iteración y acumuladores.
- Controlar flujo con `break` y `continue`.

### 08 — Funciones (`08-Python-Funciones.ipynb`)
**Objetivos:**
- Definir funciones reutilizables con `def`.
- Trabajar con parámetros, valores de retorno y alcance (scope).
- Organizar soluciones por módulos/funciones para mejorar legibilidad.

### 09 — Clases y Objetos (`09-Python-Clases.ipynb`)
**Objetivos:**
- Comprender Programación Orientada a Objetos (POO).
- Crear clases, atributos y métodos.
- Instanciar objetos y usarlos para modelar problemas.

### Práctico — Bono 1 (`Practico_Bono_1.ipynb`)
**Objetivos:**
- Integrar los conceptos anteriores en ejercicios prácticos.
- Fortalecer lógica, claridad y estructura del código.
- Practicar resolución paso a paso y buenas prácticas.
